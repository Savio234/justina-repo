# ts-utils
> Typescript utility functions

[![npm version](https://badge.fury.io/js/ts-utils.svg)](https://badge.fury.io/js/ts-utils)

## Install

```
npm install --save ts-utils
```

## API

* `assertT`

* `assignSame`
* `assignPartial`
* `assignToPartial`
* `assignSameToEmpty`
* `assignPartialToEmpty`
* `assignToPartialOnEmpty`
* `assignToDiff`
* `assignDiff`

* `allUnionTypeValues`
